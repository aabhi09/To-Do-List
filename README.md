# To-Do-List

A lightweight, terminal-based to-do list manager written in **Pure Python**.

Features persistent storage, priorities, due dates, tags, overdue highlighting, and colored output using ANSI escape codes.

## Features

- Add tasks with priority (low/medium/high), due date, and optional tag
- Mark tasks as done
- Delete tasks
- Clear all completed tasks
- List all tasks or only pending ones
- Visual overdue warnings
- Tasks saved automatically to `todo.json`
- Colorful terminal output (works in most modern terminals)
- Zero external dependencies

## Requirements

- Python 3.6+
- No pip installs needed!
